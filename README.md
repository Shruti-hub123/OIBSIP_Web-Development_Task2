👩‍💻 Shruti Mukherjee | Personal Portfolio Website

Showcasing Skills, Projects & Passion for Technology

This is a modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. The portfolio highlights my skills, projects, educational background, interests, and contact information through a clean, professional, and visually appealing interface.

The project is designed to provide visitors with a comprehensive overview of my technical journey while demonstrating responsive web design, interactive UI elements, and front-end development skills.
